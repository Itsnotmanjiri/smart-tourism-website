
  # Smart Tourism Website


## Description
A smart tourism website developed as a student project with interactive and AI-based features. It helps users explore tourist places, chat with an AI assistant, and get personalized recommendations.

## Tech Stack
- HTML, CSS, JavaScript
- React

## Features
- AI-based chatbot to answer user queries about tourist spots
- Tourist place recommendations
- User-friendly interface
- Responsive design
- carpool system
- match and social
- transalator
- currency convertor
- emergency ios

## Project Status
Completed

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
